Starting Again with Java

After spending 10 years programming and leading teams of programmers, I have decided to start learning Java all over again right from the basics by following a solid textbook. This post marks the first post in a series of posts related to the Java programming language.

Java is a versatile, object-oriented programming language that serves as a great foundation for developing cross-platform applications. If you're just starting out, learning Java's basic syntax is an excellent first step. Today, I’ll go over some core elements of Java syntax and share a book recommendation that I’ll be following myself to dive deeper into Java.

Basic Structure of a Java Program

Every Java program starts with a class definition. Here’s a simple "Hello, World!" example to illustrate the structure:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation:

  1. Class Declaration: public class HelloWorld - Every Java program consists of one or more classes, and every class has a name (in this case, HelloWorld).
  2. Main Method: public static void main(String[] args) - This is the entry point of every Java application. The main method must be public, static, and void.
  3. Print Statement: System.out.println("Hello, World!"); - The println method prints the text to the console.

Basic Data Types

Java has several basic data types. Here are some common ones:

public class DataTypesExample {
    public static void main(String[] args) {
        int number = 10;       // integer type
        double price = 19.99;  // double type for decimals
        char letter = 'A';     // single character
        boolean isTrue = true; // boolean type (true or false)
    }
}

Common Data Types:

  • int - for whole numbers
  • double - for decimal values
  • char - for single characters
  • boolean - for true or false values

Conditional Statements

Java supports basic conditional statements like if, else if, and else:

public class ConditionalExample {
    public static void main(String[] args) {
        int age = 18;

        if (age >= 18) {
            System.out.println("You are an adult.");
        } else {
            System.out.println("You are a minor.");
        }
    }
}

Loops

Loops are used to repeat a block of code multiple times. Here’s an example of a for loop:

public class LoopExample {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println("Iteration: " + i);
        }
    }
}

Recommended Books to Start Learning Java

If you're looking to delve deeper, there are several books that can help you build a strong foundation in Java:

  • Java: The Complete Reference by Herbert Schildt
  • Head First Java by Kathy Sierra and Bert Bates
  • Effective Java by Joshua Bloch

Personally, I’ve decided to follow Java: A Beginner's Guide by Herbert Schildt. It provides a well-organized introduction, covering all the basics in a beginner-friendly way. Whether you're new to programming or just new to Java, this book is an excellent resource to get you started.

Happy coding!
